- Set timeouts

- Self-preservation	
- Asynchronous messaging	
- Ping/Echo	
- Monitor	
- Heartbeat	
- Sanity Checking	
- Transactions

- Build Separation	
- Container Deployment	
- Network Location	
- Balancing Scale	
- Self-description	
- Discover Service	
- Orchestrate	
- Tailor Interface	
- Autheticate Actors
